# Docker Project Explorer

Electron Forge + React + TypeScript foundation for a polished Docker exploration app focused first on safe, read-heavy workflows.

## Prerequisites

- Node.js 22 or newer
- npm matching that Node installation
- Docker CLI available on the machine for runtime features

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Quality checks

```bash
npm run check
```

## Notes

- This scaffold follows the Webpack + TypeScript Electron Forge shape requested in the spec.
- Package versions are intentionally expressed as `latest` here because dependency resolution could not be performed from this environment.
- After `npm install`, a lockfile should be committed.
